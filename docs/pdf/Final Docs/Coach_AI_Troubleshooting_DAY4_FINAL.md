# THE 6 CRITICAL ISSUES

## Issue #1: Can't Log In / Forgot Password
**Severity:** CRITICAL (blocks all access)  
**Frequency:** 15-20% of new users  
**Resolution Time:** 2-5 minutes

## Issue #2: Practice Won't Generate / Stuck Loading  
**Severity:** CRITICAL (blocks core feature)  
**Frequency:** 10-15% of users  
**Resolution Time:** 3-10 minutes

## Issue #3: PDF Export is Blank / Won't Download
**Severity:** HIGH (prevents field use)  
**Frequency:** 8-12% of users  
**Resolution Time:** 2-5 minutes

## Issue #4: Drills Don't Match Team / Wrong Age/Skill
**Severity:** MEDIUM (quality issue)  
**Frequency:** 5-10% of users  
**Resolution Time:** 5-10 minutes

## Issue #5: Settings Won't Save / Keep Resetting
**Severity:** MEDIUM (frustration)  
**Frequency:** 5-8% of users  
**Resolution Time:** 3-8 minutes

## Issue #6: Can't Star Drills / Favorites Don't Work
**Severity:** LOW (feature limitation)  
**Frequency:** 3-5% of users  
**Resolution Time:** 1-3 minutes

### Iteration Indicators

**Add new issue if:**
- Same new problem reported 5+ times
- Blocks core functionality
- Affects 5%+ of users

**Update existing issue if:**
- Solution doesn't work for 20%+ of users
- New browser/OS causes different steps
- Better solution discovered

**Remove/demote issue if:**
- Frequency drops below 2% of users
- Technical fix eliminates problem
- Merged into different issue

# TESTING PROTOCOL

## Before Launch Testing

### Test Each Issue (2 people × 6 issues = 12 tests)

**Setup:**
1. Find someone unfamiliar with Coach AI
2. Create the problem scenario (disable pop-ups, use wrong password, etc.)
3. Give them ONLY the troubleshooting guide
4. Watch them work through solution
5. Do NOT help unless completely stuck >10 minutes

**What to Measure:**
- Time to find correct issue (should be <30 seconds)
- Time to resolve (should match estimate ±3 minutes)
- Did they succeed without help? (Yes/No)
- Where did they get confused? (note specific steps)
- What was unclear? (rewrite those sections)

**Pass Criteria:**
- 2/2 testers resolve successfully
- Average time within estimate
- No major confusion points
- Users feel confident in solution

### Integration Testing

**Test All Cross-Links:**
- Click every link to FAQ → Does it go to right question?
- Click every link to Quick Start → Does it go to right step?
- Click every link to User Guide → Does it go to right section?
- All links should open in new tab (web version)

**Test User Flows:**
1. **New user can't log in:**
   - Searches "login problem"
   - Finds Issue #1
   - Follows Quick Fix
   - Resets password successfully
   - Logs in

2. **User can't generate practice:**
   - Checks symptoms (matches Issue #2)
   - Expands issue
   - Follows decision tree
   - Identifies browser cache problem
   - Clears cache
   - Generates successfully

3. **User PDF is blank:**
   - Sees Issue #3
   - Tries Quick Fix (allow pop-ups)
   - Works immediately
   - Downloads PDF successfully
