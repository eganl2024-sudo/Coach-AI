### User Testing Protocol (45 minutes - 15 min per user)

**CRITICAL RULE:** Don't help! Just watch and take notes.

**Instructions to Give Test User:**

> "Hi! I'm testing a guide I wrote for a new app. I need you to follow this guide exactly and try to complete the task. I won't answer any questions or help - just do your best. Tell me what you're thinking out loud as you go. Ready?"

**Hand them:**
- The Quick Start Guide (printed or on screen)
- Access to Coach AI

**Your Job:**
- Sit behind them (or beside, not helping)
- Take notes on everything
- Record their screen if possible
- Don't interrupt unless they're completely stuck for 2+ minutes

## HOUR 2: REVISIONS BASED ON TESTING

### Revision Checklist

Based on your testing notes, work through these common fixes:

**□ STEP CLARITY**

If users got stuck on any step:
- [ ] Rewrite that step with simpler language
- [ ] Add more detail about what to look for
- [ ] Include a troubleshooting note
- [ ] Consider adding an extra screenshot

**□ SCREENSHOT IMPROVEMENTS**

If users couldn't find buttons/elements:
- [ ] Make annotations bigger and bolder
- [ ] Add more arrows pointing to exact locations
- [ ] Use numbers to show sequence clearly
- [ ] Highlight the element with bright color

**□ TERMINOLOGY FIXES**

If users didn't understand terms:
- [ ] Replace jargon with plain language
- [ ] Add brief definitions in parentheses
- [ ] Use examples instead of abstract terms
- [ ] Consider a glossary box

**□ SEQUENCING ISSUES**

If users skipped steps or went out of order:
- [ ] Make step numbers more prominent
- [ ] Add "Next:" transitions between steps
- [ ] Include progress indicators
- [ ] Add warnings about skipping steps

**□ EXPECTATION GAPS**

If users expected different results:
- [ ] Add "What you'll see" descriptions
- [ ] Include example outcomes
- [ ] Set expectations upfront
- [ ] Add success criteria for each step

### Revision Workflow

For each issue identified in testing:

1. **Identify the exact location** in the guide
2. **Determine root cause:**
   - Writing unclear?
   - Screenshot unhelpful?
   - Information missing?
   - Sequence confusing?
3. **Apply appropriate fix** (see examples above)
4. **Mark as revised** in your tracking doc

**Pro Tip:** Don't try to fix everything at once. Prioritize based on:
- How many users experienced it (3/3 = critical)
- How much it blocked progress (total stopper = critical)
- How easy it is to fix (quick wins first)

## 🎯 ESSENTIAL TASKS (Start Here!)

These 6 tasks introduce you to Coach AI's core features. Complete all 6 to unlock consistent, professional practice planning.

---

### ☑️ TASK 1: Create Your Team Profile (2 minutes)

**What:** Set up basic team information  
**Why:** Enables AI to generate age-appropriate, properly-sized practices

**How to Complete:**

1. Click **"Create Team"** on the home page
2. Fill in 4 fields:
   - Team Name: (e.g., "U12 Eagles")
   - Age Group: Select from dropdown (U8, U10, U12, etc.)
   - Skill Level: Beginner, Intermediate, or Advanced
   - Roster Size: Number of players (e.g., 14)
3. Click **"Save Team"**

**Success Criteria:**
✅ You see "Team created successfully"  
✅ Your team name appears in the sidebar  
✅ You can see your team in Team Hub (Advanced mode)

**Need help?** → Quick Start Guide, Step 2

---

### ☑️ TASK 2: Generate Your First Practice (3 minutes)

**What:** Create a complete 90-minute practice plan  
**Why:** Experience the core value of Coach AI - instant professional practices

**How to Complete:**

1. Click **"⚽ Generate Practice"** in sidebar
2. Configure practice:
   - Duration: **90 minutes**
   - Session Type: **● Balanced**
   - Players: (auto-filled from team)
3. Click **"Generate Practice"**
4. Wait 2 seconds for AI to select drills
5. Scroll through your practice plan

**Success Criteria:**
✅ Practice appears with 5-6 drills  
✅ You see warmup, technical, games, cool down  
✅ Intensity curve shows proper progression  
✅ Total time = 90 minutes

**Need help?** → Quick Start Guide, Step 3

---

### ☑️ TASK 3: Browse the Drill Library (5 minutes)

**What:** Explore 45+ professional soccer drills  
**Why:** Understand what's available and find drills you like  
**Note:** Requires Advanced or Expert mode

**How to Complete:**

1. Upgrade to Advanced mode (sidebar → Experience Level → Advanced)
2. Click **"🏀 Drills"** in sidebar
3. Browse through drills (scroll or filter)
4. Click on 3-5 drills to read details
5. Note the different categories, intensity levels, player counts

**Success Criteria:**
✅ You see the full drill library (45+ drills)  
✅ You understand drill card information  
✅ You've read 3-5 complete drill descriptions  
✅ You recognize the filter options

**Need help?** → User Guide → Drill Library section

---

### ☑️ TASK 4: Star Your First Favorite Drill (1 minute)

**What:** Mark a drill as a favorite  
**Why:** Build your personal library of proven drills for quick access

**How to Complete:**

1. In Drill Library, find a drill you like
2. Click the **☆** star icon on the drill card
3. Star turns gold: **★**
4. Filter to "Favorites Only" to see your starred drills

**Success Criteria:**
✅ At least 1 drill is starred  
✅ Star icon is gold ★  
✅ Drill appears when filtering to favorites  
✅ You understand how to unstar (click again)

**Need help?** → User Guide → Favoriting System

---

### ☑️ TASK 5: Review Practice History (2 minutes)

**What:** See all your saved practices  
**Why:** Access past practices and learn the reuse workflow

**How to Complete:**

1. Click **"📅 History"** in sidebar
2. See your list of saved practices
3. Click **"View Details"** on your first practice
4. Explore the detail view
5. Return to History page

**Success Criteria:**
✅ You see at least 1 saved practice  
✅ You can view practice details  
✅ You understand the date/duration/type display  
✅ You see the "Reuse" button (for next task)

**Need help?** → User Guide → Practice History

---

### ☑️ TASK 6: Customize Your Settings (3 minutes)

**What:** Set your default preferences  
**Why:** Practices will start with YOUR preferred settings every time

**How to Complete:**

1. Click **"⚙️ Settings"** in sidebar
2. **General tab:**
   - Default Duration: Select your usual practice length (e.g., 90 min)
   - Default Session Type: Choose most common (e.g., Balanced)
   - Auto-save Practices: Turn **ON**
3. Click anywhere outside settings (auto-saves)
4. Generate a new practice to verify defaults

**Success Criteria:**
✅ Default duration is set  
✅ Default session type is set  
✅ Auto-save is enabled  
✅ Next practice generation uses your defaults

**Need help?** → User Guide → Settings → General

## 🔥 ADVANCED TASKS (After Completing Essentials)

Once you've completed tasks 1-6, level up your skills with these advanced workflows:

---

### ☑️ TASK 7: Complete Your Team Profile (5 minutes)

**What:** Add formation, play style, and focus areas  
**Why:** AI will prioritize drills that match YOUR team's specific needs

**How to Complete:**

1. Navigate to **Team Hub** (sidebar)
2. Click **"Edit Team Profile"**
3. Add the following:
   - **Formation:** (e.g., 4-3-3, 4-4-2, 3-5-2)
   - **Play Style:** (e.g., Possession, Counter-attack, High Press)
   - **Focus Areas:** Add 3-5 skills you're working on
     - Example: "First touch", "Quick passing", "Building from back"
4. Save profile (100% completion!)
5. Generate a new practice to see drill matching

**Success Criteria:**
✅ Profile completion shows 100%  
✅ Formation and play style are set  
✅ 3-5 focus areas added  
✅ Next practice shows "Matched focus: [area]" labels

**Why This Matters:**
With focus areas set, Coach AI prioritizes drills that develop those specific skills. This creates targeted, developmental practices instead of random drills.

**Need help?** → User Guide → Complete Team Management

---

### ☑️ TASK 8: Reuse a Successful Practice (1 minute)

**What:** Copy a past practice for instant planning  
**Why:** THE FASTEST WORKFLOW - 1 minute to a complete practice!

**How to Complete:**

1. Go to **Practice History**
2. Find a practice you liked
3. Click **"Reuse"** button
4. You're taken to Practice Generator with drills pre-selected
5. (Optional) Modify date or settings
6. Click **"Generate Practice"**
7. Same drills, new date - done!

**Success Criteria:**
✅ You successfully reused a practice  
✅ Same drills appeared in new practice  
✅ Total time was under 1 minute  
✅ You understand when to use this workflow

**Pro Tip:** This is the secret weapon for weekly planning. Find 2-3 great practices and rotate them throughout the season with variations.

**Need help?** → User Guide → Practice History → Reuse

---

### ☑️ TASK 9: Upgrade Experience Level (30 seconds)

**What:** Try Advanced or Expert mode  
**Why:** Unlock more features and customization options

**How to Complete:**

1. Click **Experience Level** dropdown in sidebar
2. Select **"Advanced"** (or "Expert" if you want ALL features)
3. Interface updates to show new menu items
4. Explore new pages:
   - Advanced: Drills, Team Hub, Favorites
   - Expert: Templates, Cycles, Weekly Planner (future)

**Success Criteria:**
✅ Experience level changed  
✅ New menu items visible in sidebar  
✅ You can access previously hidden features  
✅ You understand you can downgrade anytime

**Decision Guide:**
- **Essential:** Keep it simple, fast planning
- **Advanced:** You want customization and drill library (most popular)
- **Expert:** You're a power user who wants templates and automation

**Need help?** → User Guide → Understanding Experience Levels
